Steps that worked - How to clone a Jekyll blog site, as Steps shown on this Video link

    1/ create a new REPOSITORY on Github with Readme.md option check, say Notes
    2/ go to Desktop Github client and clone above Notes
    3/ on a separate directory on Terminal, git clone desired Github blog
    4/ copy all files ( -r) from dir in Step 3 into Step 2 dir, create a .gitignore file with 3 entries since this is NOT copied
    5/ on Desktop client , commit all files then Sync to Notes Git branch
    6/ go to Github Notes REPOSITORY on Browser, clone master branch as gh-pages
    4/ set gh-pages as Default branch, then hit URL
